Программа позволяет загрузить,распаковать и протестировать работы студентов, загрузивших ответы на задания на ресурс
с развернутым Moodle. Работа программы была проверена на системах windows, а так же ubuntu подобных Linux системах.

Для корректной работы программы необходим настроенный сервер.
Необходима установленая php7 и curl. Для тестирования работ должны быть установлены такие утилиты, как:
1. Cmake
2. QT 5.5
3. Make

Для того, что бы иметь возможность отправить результат на почту, следует
установить и настроить один из агентов передачи почты. Например, Sendmail.
В файле conf.ini происходит настройка программы для работы.

Следует указать в файле с настройками:
username - логин пользователя на ресурсе с развернутым moodle
password - пароль пользователя на ресурсе
login_url - страница для авторизации в moodle
task_url - пример страницы с заданиями (без указанного id, например "http://edu.vstu.org/mod/poasassignment/view.php?id=")
task_id[] - задания, необходимые для проверки, могут быть указаны несколько заданий на отдельных строчках

Если требуется отослать результат проверки, следует указать опцию send_result_on_email
send_result_on_email   - true, для того что бы результат был отправлен на почту
email - адресс, куда следует отправить результат

Если требуется скачать и сохранить работы студентов следует указать опцию
save_answers  - true , для сохранения ответов
files_download_to - папка в которую будут сохранены работы

Если требуется распаковать скаченные файлы следует указать опцию
unpack_answers - true
path_to_winrar - путь к установленному распаковщику для распаковки архивов

Если требуется протестировать скаченные работы следует указать опцию
build_and_compile - true

Так же необходимо указать путь к cmake, qmake, make
path_to_CMake 
path_to_QMake 
path_to_Make 

Следует указать какой генератор должен использовать cmake при генерации
generator_for_CMake

Если работа происходит в linux системах, пути указывать не следует.
